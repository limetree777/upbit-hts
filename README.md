# upbit-hts

https://wikidocs.net/book/1665   
위사이트를 참고하여 만드는중인 업비트 거래 프로그램입니다.   

## 현재 기능   
-시세조회   

## 추가할 기능   
-호가창   
-매수/매도   
-키보드로 빠르게 매수/매도   

## 문제점   
-웹소켓으로는 한번에 여러 종목의 가격을 표시할수 없다   
웹소켓이 초당 5개밖에 가져올수없어서
